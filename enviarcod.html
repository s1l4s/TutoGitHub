Enviando o código para o repositório no GitHub

mkdir "nome da pasta"
cd "nome da pasta"
git init
git status
git add "nome do arquivo"
git commit -m "comentário"
git remote add origin "link do repositório"
git push -u origin master

Enviando alterações no código para o repositório no GitHub

git status
git add "nome do arquivo"
git commit -m "comentário"
git push
